## Тема
B2B Marketplace
## Проблема
Проблема "Упрощение работы маркетплейса для пользователей, вся нужная информация находиться перед пользователем"
## Объект и предмет
Объект: [Разрабатывается удобный интерфейс и добавляются удобный функции для пользователей]
Предмет:Интерфейс и функции 
## Цель
Создать дизайн и описание макета B2B-маркетплейса для компаний, которые занимаются оптовыми закупками.
## Основные задачи
1.[Реализовать функции в интерфейсе, для удобства пользователей]
2.[Реализация удобного интерфейса]
3.[Реализация упрощения работы внутри маркетплейса]

## Ожидаемый результат
[Реализованная программа B2B Маркетплейса, интересный и удобный интерфейс для пользователей]
