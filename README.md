# B2B Маркетплейс

## Назначение проекта
Данная курсовая работа посвящена разработке прототипа интерфейса B2B-маркетплейса для компаний, которые закупают товары оптом.

Проект должен показать, как можно упростить поиск товаров, просмотр информации о поставщиках и отправку заявки на закупку.

## Состав репозитория
- docs/ — проектная документация и методика;
- src/ — исходные файлы, прототипы или скрипты;
- data/sample/ — небольшой пример данных;
- experiments/ — журнал проверок, расчётов или экспериментов;
- assets/ — изображения, схемы, диаграммы.

## План выполнения
1. Постановка задачи и анализ предметной области.
2. Подбор источников и требований.
3. Разработка макета и описания интерфейса.
4. Проверка результата.
5. Подготовка пояснительной записки.

## Текущий статус
Сформирована постановка задачи, создана структура репозитория и начата работа над методикой проекта.
